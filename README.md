# Global-Ecommerce-Sales-Analysis
Data Analytics Internship Project using Excel, SQL, and Power BI.

## Project Overview

This project was completed as part of my Data Analytics Internship. The objective was to analyze global e-commerce sales data and generate actionable business insights using Excel, SQL, and Power BI.

## Tools Used

* Microsoft Excel
* MySQL
* Power BI
* GitHub

## Project Workflow

1. Data Cleaning using Excel
2. SQL Analysis using MySQL
3. Business Insights Generation
4. Interactive Dashboard Development using Power BI

## Key Metrics

* Total Sales: 12.64 Million
* Total Profit: 1.47 Million
* Total Orders: 51.26 Thousand
* Average Discount: 14%

## Dashboard Highlights

* Sales by Region
* Sales by Market
* Monthly Sales Trend
* Category Performance
* Top Products
* Top Customers
* Segment Analysis
* Profit by Sub-Category

## Business Insights
The Global E-Commerce business demonstrates strong financial performance, generating 12.64 Million in sales and 1.47 Million in profit. The APAC market and Technology category emerged as major growth drivers, while discount optimization and improvements in low-performing regions present opportunities for future growth.
The combination of Excel for data preparation, SQL for analysis, and Power BI for visualization enabled meaningful insights that support data-driven business decision-making.
